# WinSettingsGui

Tray-приложение для Windows 11 для быстрого управления настройками питания (экран, сон, гибернация) через вложенное контекстное меню.

## Возможности

- Управление таймаутами отключения экрана (AC/DC)
- Управление таймаутами перехода в сон (AC/DC)
- Управление таймаутами гибернации (AC/DC)
- Автоопределение текущих системных настроек при запуске (галочки в меню соответствуют реальным значениям)
- Настраиваемые значения таймаутов через диалог настроек (применяются без перезапуска)
- Конфигурация сохраняется в `%APPDATA%/WinSettingsGui/config.json`
- Автозапуск при старте Windows (через реестр)
- Иконка в системном трее с вложенными меню
- Поддержка русской и английской локализации Windows (парсинг powercfg)

## Структура меню

```
Настройки питания
├── Экран
│   ├── При питании от сети
│   │   ├── Отключать через 1 мин  [✓]
│   │   ├── Отключать через 5 мин
│   │   ├── Отключать через 15 мин
│   │   ├── Отключать через 30 мин
│   │   └── Не отключать экран
│   └── При питании от батареи
│       └── (аналогично)
├── Спящий режим
│   └── (аналогично)
└── Гибернация
    └── (аналогично)
───────────
Настройки         → диалог настройки таймаутов
Автозапуск    [✓] → переключатель
О программе       → информация
───────────
Выход
```

## Технологии

| Назначение | Библиотека |
|---|---|
| Системный трей | `github.com/energye/systray` |
| Диалог настроек | Win32 API (CreateWindowEx) |
| Диалог "О программе" | `MessageBoxW` через `syscall` |
| Автозапуск | `golang.org/x/sys/windows/registry` |
| Управление питанием | `powercfg` через `os/exec` |

## Конфигурация

Конфиг хранится в `%APPDATA%/WinSettingsGui/config.json`. По умолчанию таймауты: `[1, 5, 15, 30]` минут для каждой категории.

## Сборка

```bash
go build -ldflags "-H=windowsgui -s -w" -o WinSettingsGui.exe
```

## Использование

1. Запустить `WinSettingsGui.exe`
2. В системном трее появится иконка
3. ПКМ по иконке — контекстное меню с настройками питания
4. Выбор таймаута — мгновенное применение через `powercfg`
